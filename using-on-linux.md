# Using on Linux

## Overview

RAD-Meteor enumerates as a NIC on Linux. Plug in and use like any other Ethernet Interface. The RAD-Meteor will be listed in "ifconfig" with a MAC address starting with Intrepid's OUI (00:FC:70).
